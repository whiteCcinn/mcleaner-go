- 需要配置的文件 (https://github.com/spf13/viper)

- 支持中间存储失败的数据（需要持久化）(https://github.com/dgraph-io/badger)
- 打包工具 (https://github.com/goreleaser/goreleaser)
- cli (https://github.com/spf13/cobra)
